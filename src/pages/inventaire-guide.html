📦 Guide d'utilisation - Inventaire

Bienvenue ! Ce guide vous explique comment utiliser les nouvelles fonctionnalités d'inventaire.

🎯 Vue d'ensemble

La section Inventaire du tableau de bord vous permet de :

  • ✅ Ajouter des articles un par un via le formulaire
  • ✅ Remplir rapidement des articles directement dans le tableau
  • ✅ Importer des données depuis un fichier Excel ou CSV
  • ✅ Exporter l'inventaire complet en CSV
  • ✅ Rechercher et filtrer les articles

🚀 Fonctionnalités

1️⃣ Afficher/Masquer le tableau (Clic sur le titre)

Cliquez sur le titre "📦 Inventaire" pour afficher ou masquer le tableau complet.

Raccourci : Les contrôles et le tableau n'apparaissent que si vous cliquez sur le titre.

2️⃣ Remplissage rapide (Ligne au bas du tableau)

Une ligne spéciale gris clair pointillée se trouve au bas du tableau.

Comment l'utiliser :

  1. Remplissez les champs : ID, Nom, Catégorie, Marque, Quantité, Lieu
  2. Cliquez sur ✅ Ajouter ou appuyez sur Tab dans le dernier champ
  3. L'article s'ajoute immédiatement, et les champs se réinitialisent

Raccourcis clavier :

  • Tab dans le dernier champ → Ajoute l'article
  • Entrée dans le dernier champ → Ajoute l'article
  • Clic sur 🔄 Effacer → Réinitialise tous les champs

3️⃣ Ajouter un article (Bouton "+ Ajouter article")

Ouvre un formulaire modal complet pour ajouter un nouvel article.

Quand l'utiliser : Pour une saisie plus détaillée ou contrôlée.

4️⃣ Importer depuis Excel/CSV (Bouton "📥 Importer Excel/CSV")

Permet d'importer plusieurs articles à la fois depuis un fichier.

Étape 1 : Préparer le fichier
Téléchargez le modèle CSV : 📋 Modèle CSV
Le fichier doit avoir ce format :
ID Nom Catégorie Marque Quantité Emplacement
P001 Clavier USB Périphériques Logitech 25 45.50 Rayon A Clavier standard
P002 Souris sans fil Périphériques 30 28.00 Rayon A Souris optique
Étape 2 : Importer le fichier
1. Cliquez sur 📥 Importer Excel/CSV
2. Sélectionnez votre fichier .csv ou .xlsx
3. Confirmer l'import → Les articles seront fusionnés avec l'inventaire existant
Les articles avec un ID déjà existant seront mis à jour.
Les nouveaux articles seront ajoutés.

5️⃣ Exporter l'inventaire (Bouton "📤 Exporter (CSV)")

Télécharge un fichier CSV avec tous les articles actuels.

Utilité : Sauvegarde, envoi par email, ou édition dans Excel.

⚡ Conseils d'utilisation

Pour la saisie rapide : Utilisez la ligne au bas du tableau pour ajouter des articles individuels. C'est plus rapide que le formulaire modal.
Pour les imports massifs : Préparez votre fichier Excel, exportez-le en CSV, puis importez-le. Utilisez le modèle fourni pour avoir le bon format.
Format CSV important : Assurez-vous que votre fichier CSV a une en-tête avec ces colonnes : ID, Nom, Catégorie, Marque, Quantité, Emplacement
Automatisation : Tous les ajouts sont automatiquement loggés pour l'audit. Consultez la section "Logs et audit" pour voir l'historique complet.

❓ Dépannage

❌ "L'ID existe déjà"
Vous avez essayé d'ajouter un article avec un ID qui existe. Utilisez un ID unique ou modifiez l'article existant.
❌ "Veuillez remplir tous les champs"
Tous les champs de la ligne de saisie rapide sont obligatoires. Vérifiez que aucun n'est vide.
❌ Import échoué : "Aucun article valide trouvé"
Votre fichier CSV n'a pas le bon format. Téléchargez le modèle et copiez vos données dedans.
